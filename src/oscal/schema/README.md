# Schema Directory

This folder contains the LinkML schema YAML files for the OSCAL models in this
repository.

Current standalone schemas:

- `oscal_catalog.yaml`
- `oscal_assessment_plan.yaml`
- `oscal_assessment_results.yaml`
- `oscal_component.yaml`
- `oscal_mapping.yaml`
- `oscal_poam.yaml`
- `oscal_profile.yaml`
- `oscal_ssp.yaml`
- `oscal.yaml`
